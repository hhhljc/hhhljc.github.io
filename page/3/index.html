一周奥兰多

发表于 2018-01-17

AIAA会议

我不是外向的人，但我需要适应做报告，所以要努力。
事实上，在去之前我花了大量的时间练习，临行前的组会，老师照例让每个人都做了演练，相应提出改进意见，后来还专门发邮件对slides做了具体修订。
当时主要的问题有三点：

  1. 英语表达仍然欠缺
  2. 不够自信，需要更频繁地与观众进行眼神交流
  3. 肢体语言不够。

总结
并没有拿到最佳论文，不重要；最欠缺的是socialization，本应做得更好。
期间有各地的牛人，还有一些非常有意思的报告。
听了很多场，却没有一次提问，几乎也没有跟作报告的人进行交流，只是去找了一个国内来的学弟瞎聊了几句没营养的。
事实上，其中的一个报告者就是我后来一篇文章的审稿人，他给了一些很有建设性的意见。
虽然文章后来被顺利接受，但当时能去和他聊一下自己的想法，这个过程必将会让自己对social的认识更为积极。
而这种认识是必需的。

新奥尔良 — 一路向南五日游

发表于 2017-12-30 | 分类于 旅行

没几天就要去佛罗里达了，还有不少工作要准备，本不该选择此时出游，况且同是南方。无奈经不住自驾的诱惑，还是有了此行。

行程

人物

高冷哥，热情弟，寡言的前舍友“未成年”君，我。

感悟

  • 没做过的事情不一定都要做，要心存敬畏，尊重习俗／习惯

    对宗教本身没有偏好，但对教堂里的活动感兴趣，一直想找机会进去体验一下。今天真巧，路过圣路易斯大教堂，刚好在举行活动，人很多，还有盛装的宗教人士领读／唱，穿得跟教皇一样。后来看大家排队到前面接受司仪(?)的什么仪式，我和热情弟跃跃欲试，但又怕有失尊重。后来还是基于‘不做会后悔／不疯狂就老了’的逻辑跟了上去。轮到我了，“司仪”拿出一片薯片之类的东西，我不知道该怎么接，刚想伸手，就看他皱起眉头，轻轻摇了摇头。我尴尬得不知所措，他的表情也愈加凝重。后来我才猜出来大概是要用嘴去接的…

    Shame on myself! 后来我一直在想，对自己的不满到底来自哪里。对宗教的敬畏？还是对自己唐突的鄙视？

    不会忘记的是那个略带愠怒和宽容的眼神。我无意挑战任何人的底线，也不想我行我素后祈求宽容。我感兴趣的是真诚的赞赏、由衷的掌声。这一生没做过的事儿会有很多，但每一颗好奇心的满足都要建立在尊重别人的基础上。

  • 自己的权益自己争取

    当地有很多农奴时期留下的种植庄园，我们去的橡树庄园，算是最大的一个。售票员是位白发苍苍的奶奶，聊了两句后，我们准备刷卡买票，4个人大概一百美元。她说好像电脑有点故障，让稍微等一下，我们也不急，就在旁边聊天。大概过了3分钟，她说不好意思，还得等一会儿，她让IT guy过来处理了。我们觉得无所谓，这时高冷哥突然问了一句“Is there a waiting discount?”, 老奶奶干脆的回了一句“Yes, you have to pay more!” 我们大笑。两分钟后，电脑好了，老奶奶说“You get free admission, just go!” 我们便免票入园了，留下新来的一群人在风中凌乱。

    高冷哥得意得说“自己的权利自己不争取，还等着别人主动给你吗？” 对高冷哥肃然起敬了。有枣没枣，敲两杆子试试。Why not？

  • 不能完全信赖任何人

    热情弟是个靠谱的人，高冷哥也是。但也有马失前蹄的时候，比如去看海，开了将近两个小时却没能找到入海口，看到的只是大片的沼泽地。

    还是有必要review一下向导的计划的，即便我们只是“负责开心”。

底特律半日游

发表于 2017-11-29 | 分类于 旅行

半日底特律之行已是大半个月前的事儿了，刚刚翻了翻照片，发现那天竟是双十一，好巧。底特律算是离学校最近的“大城市”，早已列入计划，想去见识一下荒凉，却迟迟没定下行程。天气渐渐转凉，再不去真就要冬眠了。某天中午刚好在餐厅碰见朋友，一拍即合，定下了周末去转转。交通很方便，学校在底特律downtown设有一个交流中心，每天有通勤班车，往返$16。觉得也是半主场，直到周五晚上才粗略看了看攻略。预定了车票，周六早上8点半从中校出发，下午4点返回。但到睡觉前才发现，周六早上并没有去中校区的公交！只能试着发邮件改时间，还真可以，于是实际的半日游是从下午1点到晚上8点。

人穷更该会花钱

发表于 2017-11-22 | 分类于 感想

拿5%的钱投资自己，这篇文章讲的颇有道理。想想参加一些培训班也是一件很有意思的东西，比如去学陶艺。那么终极之问又来了，意义在哪？一件稍稍有趣的事情，如果不确定是否有意义，那么去做至少还有能排除它有意义的意义。 :-1:

有几点深表赞同：

  • 做细致的白日梦
  • 参加兴趣班
  • 尽量不要拿时间换金钱。比如应该住在离公司较近的地方，哪怕贵点。好好利用省下来的时间。
  • 如果真的入不敷出，拿更高比例的钱投资自己 😱

中美共同努力促进双向留学

发表于 2017-11-09 | 分类于 时事

特朗普访问中国，达成创纪录的2500亿美元的经贸合作协议。同时双方达成多项共识，其中第八项是:

双方将共同努力促进中美双向留学

双方将共同努力促进中美双向留学，推动两国教育机构和学者间的交流合作。
中方将实施“优秀美国学生短期留学中国计划”，未来4年向美国提供一万个学分生奖学金名额。
双方欢迎每年在两国举办“中美青年创客大赛”，并在两国新设若干“中美青年创客交流中心”。

看到这个，心中五味杂陈。今年美国实施了更为严厉的Expert Control计划，限制了多所中美高校间的访学交流，有同学受此影响未能抵美。如果这项共识出于中方的努力，不得不说国家真是明察秋毫，点个赞！
