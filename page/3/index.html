走歌人

发表于 2020-06-03 | 分类于 音乐

什么样的歌听着能流泪？走歌人

人生如歌。
有人总是后知后觉，唱完第一句，已然半生；
辗转踟蹰，孑然前行，含泪吟唱莫愁前路无知己；
江湖飘零，哪能时时遂心，恍恍惚惚得过且过吧；
四句唱完，年华不再，穷途末路，人生就这样了吗。
才不是的，旧时友、江南秋、感慨世事难休、含泪远望独自行走。

这么多意象放在一起，矫揉造作吧，但是抵不过旋律动人，我听到那个第二句就哭了，还是很赞啊。搜了下暗杠，原来是名斗鱼主播，人称杠姐，真是英雄不问出处。知乎上对他评价很高，坚持写歌，前几年好几首很火的网路歌曲（比如童话镇）都是他的。说实话，这种歌词的作文，高中同学中也有人写得出来，但不知他们现在怎样了，希望我们都能在喜欢的领域坚持下去吧。

网易云和YouTube上都有这首歌现场版视频，很热闹很欢乐。再次鄙视下曾不理解演唱会的自己，白眼。

学农也挺好的

发表于 2020-05-30 | 分类于 感想

二十来岁的男孩大多喜欢军事科技，我也不例外。但今年特别爱看农业节目，可能是年龄大了，体内的小农基因逐渐起了作用。像央视农业频道的《致富经》、《科技苑》等，每期一个案例，故事发展一般有点起伏波折，但最后却能年入百千万，很诱人，真想搞个自己的大棚、果林或鱼塘。

我觉得诱惑可能来自这些方面：

  • 土地里谋生活，可谓是无本万利，像是拥有了传说中的摇钱树—满足感。
  • “养鱼在养水”，人在自己的一亩三分地里扮演了小生态圈内造物主的角色—成就感。

当然我很清楚，这些节目只是成功个例，普通人如果拍脑袋跟风，大概率会赔掉老婆本。我家在豫东平原，本是天下粮仓，前些年田间却冒出很多养鸡大棚，远远望去蔚为壮观。一个批鸡三千只，60天，顺风顺水能赚三两万，行情不好就要赔掉一两万。养殖户辛苦一年，养出三两批，年底算账却大多白忙活。当时我感觉搞养殖就是个赌，和种地一样，得靠天吃饭。父亲经营这行多年，对这些风险十分清楚，但也有头脑发热的时候。2009年，他在附近农场包了五个养蚕大棚，进购了一万五千只鸡苗，请了亲戚朋友来帮着管理。本以为蚕棚换茬养鸡，病害少，没想到却凶狠异常，死伤严重。最后凭着和收购商签订的保底价，才勉强保住了本钱。央视的那些故事，大概也有人跟风吧，大概最终也赔了吧。

最近，央视不再只讲述成功故事了。他们出了个新节目《田间示范秀》，先讲述失败的故事，再由帮小妹或小哥出面寻找能人帮助，最后皆大欢喜。小妹小哥不太像央视记者，非常亲民，也很卖力，因此节目内容十分丰富，时有意外的惊喜，我觉得很好看。

想做事情，得有科学的态度，学习、琢磨、实验，不怕失败。以种植为例，关键因素就是温度和湿度，大多数问题都是由这两点导致的。一个不恰当的比喻，就像数值模拟中的算法和代码，计算结果和实验对不上，不是算法有问题就是程序有bug。遇到问题不要急，也不用烧香拜佛，肯定是有些变量引发了关键因素的变化。

看了这些节目（比如20200430，20200508，20200526等），还有以下感悟。

  • 失败时受冷嘲热讽，成功后被雨露均沾。人生道路千万，故事起伏相似，小天地也能见大世界。
  • 一个人的梦里要能容下很多人的梦，才能成大事。做事本质又在做人。

老美又关上一扇门

发表于 2020-05-30 | 分类于 时事

今天周六，晴天，起床后看到朋友的推送，心情沉重。特朗普以中国“军民融合”战略为借口，颁布了中国研究人员的入美禁令，针对研究生及以上层次，可能受影响的中国高校和研究所列表包含了几乎所有211高校、部分地方高校和军工类研究所。

我不懂政治，又一向悲观，遇事容易往坏处想。美国如果严格执行该禁令，那是真要在科技上封锁中国了，这与以贸易战围堵中国的政策一致，中美矛盾将愈发不可调和。作为非建制派，特朗普一向不按套路出牌，有时候却能大破大立，对中国真是麻烦。17年底他访华的时候，中国给予了超级礼遇，他表面和气称赞，回去就大变脸，我呸。

我们该怎么办，国家层面自有考量。不过仍想大放厥词一番，我觉得可能得

  • 减少公派留学人员名额。公派美国的名额不必全数转至其他国家，中国需要的技术和美国高度重叠，因此美国经验对我们最为有利。其他国家的优势学科相对不够全面，在很多我们亟需的工程技术方面不如国内的情况比比皆是，要做好筛选。
  • 扶植发展国内学术期刊，尤其是中文期刊。目的不是搞对抗，但一旦被隔离在他们的学术圈之外，我们没必要持续向他们输送技术文章。况且期刊发展需要时间，要以十年树木的远见打提前量，十年之后我们超过美国的技术方向绝对不止5G等个别方向，完全有必要发展自己的交流舞台。
  • 改革提升国内学术环境，扩大国内学术交流活力，吸引优秀学生在国内开展研究。

第三点好难好难，中美学术环境差距很大，路漫漫其修远兮。

横穿羌塘--->《七十七天》

发表于 2020-05-13 | 分类于 电影

偶然看到一个影评，特地看了这部电影。最近对77这个数字比较有好感，前几天选了77个翼型效果挺好。关于西藏的电影我都比较感兴趣，之前的高反滋味记忆犹新。“去他妈的命运”，那必须要看一下了。

  • 很多人第一次站到雪山面前都有想哭的冲动，我之前没有，今天有。那个镜头在加乌拉山口拍摄，可能是珠峰辨识度太高了，看到后鼻子突然一酸。

  • 黛青塔娜《寂静的天空》很好听，虽然只引用很短的一段。前阵子迷王洛宾，听了一些新疆民歌，节奏大多欢快，一听就知道是瓜果飘香的地方。青藏高原寒冷空旷，不宜歌唱盛世安康，人们兀自转山朝圣，歌声清澈空灵。

  • 那个天空之境的地方真是震撼，本以为是特效做出来的，不然就是去玻利维亚拍的，没想到是真镜头。可惜在羌塘，辛亏在羌塘。

  • 看完后发现还有导演解说的同步音轨，听着又重头看了一遍。导演讲了一些拍摄背后的故事，颇有点看纪录片的感觉，如果大家都听这个给电影评分，这部片子不至于才6.4。

  • 突然想到了做研究写文章，研究内容是一个平面，文章只是这个平面中的一条线，起点到终点，简单明了，却不包含任何历史轨迹和路标，后来者难以按图索骥洞悉事情发展的全貌。

  • 只看电影还不够，还要听听导演怎么说；只看文章也不够，还要和作者聊一聊。

《王鼎钧回忆录》

发表于 2020-04-24

这一周读了王鼎钧四部回忆录的前三部，可谓是废寝忘食，感慨良多。

读完昨天的云，顿起故乡之思。从大学离家，我基本没怎么回过老家，尤其16年后，只是回了一趟齐老。我老家在豫东，鼎公家乡是临沂，在鲁南，位置相距大致不远。从文中看，“白面好吃磨难挨”，谚语也大致相同。上网查找，看到一些提议豫东皖北苏北鲁南合并一省的说法，有人评价说这些都是赤贫之地，不觉莞尔，却更拉近了和鼎公的距离。在文末，鼎公写道“少年离家还是太仓促，一口气走出五里路才回头，兰陵已不见，应该匍匐着身躯，拿着放大镜，一寸寸看”。隐约觉得他应是再无回乡的机会了，我突然有个念头，下个假期要到这些地方走一趟，替人凭吊一下故土，也喝一口山东的羊肉汤。

那是个周日的下午，窗外阳光明媚，心情却相反。想继续把其他后续三部找来看完，无奈kindle邮箱总也接受不到。另外，几日卧床读书下来，右眼痒肿难忍，还是歇息一下，也得赶赶报告了。无意中看了一个王洛宾的节目，很有兴趣，把他的歌曲搜来逐一播放，大多耳熟，又起童年之思。知乎网友开玩笑说王洛宾仅次于骆宾王，哈哈。上几代中国人的命运大多坎坷，大人物也难以幸免。令人佩服的是那首节奏欢快的青春舞曲是王洛宾在刚离婚后写下的。据说是1938年，王洛宾从西宁赶到兰州解决家庭矛盾，妻子已和他人一起生活，曾经的家自己却无栖身之地，几天后两人登报离婚。国破妻散，境况想也凄凉，写下的作品怎还能那么欢快，像首2000年左右的流行歌。

好不容易后几部才推送过来，每天也只能抽空看一点，一是眼疼，二是科研不顺，得多花心思。周五晚上交了报告，简单吃点晚饭，洗干净了爬上床，急不可耐地读起来。第二部讲求学，不似少年那般欢快，却也多新奇。国立二十二中，校长李仙州可以青史留名了。越过封锁线求学，同学都可谓心有不俗之志，然而也有置众托于不顾将邮费挥霍一光的人，有为出风头掀起学潮的人。每个时代都有他的局限性，每个同学也一样，尤其是被冲动裹挟的人流，人多的地方不要去，有道理。因此要有自己的想法和坚持，如下一部中印象最深的一句话：生活没有眼睛，欺骗是它的导盲犬。我们总会被骗来骗去，因此抓住你信的，尽管也不知真假。

学校西迁，小鼎钧一个人走了大半路程，想来又可怜又可笑。其中有一段，“进入泌阳县境，这才觉得真正置身河南，我是说天阔地广的河南，坦荡舒展的河南，古意泱泱的河南，不只为什么，我一步步走在河南的土地上觉得这才是真正的中国，直到今日，在我的回味中，仍有踩在黄色绒毯上的那种感觉。”看完很为故乡骄傲，我想王鼎钧如果是河南人，也能写出同样精彩的昨日的云吧。小时候时常在某个阳光斜照的下午庆幸自己生在了这个省这个县这个村这个家，世界上有几十亿人，我却那么幸运地占取了那几十亿分之一的概率，我真是个幸运儿！长大后走出来，渐渐不为家乡骄傲了，他们说河南人很坏，豫西人说豫东才坏。再后来，原生家庭的概念在网上火热，多少文章痛书原生家庭给自己带来的苦难。夜深反侧，渐渐也不为那个小概率而自豪了。不谈责任而顾影自怜，我们真该羞愧。

西迁路上本是和二表姐搭伴，无奈扁平足走不快，表姐不能掉队，王鼎钧散步到南阳被后一队追上又撇过。书中没有半点抱怨，真的是没有吗？我想我是会有的吧。胸襟开阔总不是坏事情，多加修炼。

第三部比较沉重，动情之处颇多，潸然泪下常有。安身立命，多有贵人相助，不然怎有后来的传奇。不能封锁自己，不然注定是个小天地。有专长，有思想，敢表达，重情义。当然他也被人利用，招致恩人反感，比如那一千发机枪子弹。但不可因噎废食。

最近要准备一篇报告，最后一篇留着之后读。

全书中印象深刻的一些话：

  • 生活没有眼睛，欺骗是它的导盲犬
  • 伤感、纤巧难成大器
  • 人切忌顾影自怜

我的2019

发表于 2020-01-25

时间轴

  • 1月 完成预答辩+论文送审。和Martins那边合作了一篇关于data-driven constraint的论文，这个月投了出去。
  • 2月 和家人一起度过在西安的第一个春节，这一年西安的夜景格外美丽，开心。月底花一个星期写了博新计划的申请书，那天下午送完材料，坐在公交车上望着窗外，觉得格外踏实、舒服，找回了美帝艰苦岁月时期的心安。
  • 3月 正式答辩，毕业了。腾出手来终于把MassiveOpt论文投出去了。
  • 4月 参加了毕业典礼，我妈也穿上了博士服，她很开心。这个月写了两个申请书—特别资助基金和重点实验室基金。
  • 5月 有一天突然腰疼得下床都很艰难，师弟久病成医，断言定是腰突了（腰间盘突出）🦉，红会医院的大夫说不确定，建议先回去多拉伸，过几天不减轻再来检查，后来果真自己就好了。这个月有了一点工资收入，算是很重要的人生第一次了，很开心地请同学和家人吃饭。记得大二的时候追女生聊将来，毫不掩饰自己对钱无感，人生目标也就是个月入三五千，女生很担忧地看着我，问该不该怀疑我挣钱的能力… 好傻
  • 6月 特别的一个月。Data-driven constraint论文顺利接受了，恰好是Martins的第100篇期刊论文，他专门祝贺了一下；特别资助中啦，开心的不单单是钱，更欣慰的是学术界专家对自己研究想法的肯定；课题组召开了一次学术会议，见了新加坡的博后导师，聊得很开心；新加坡工作签证下来了，出去留学再不存在什么变数；做了第一次中文报告，感觉不如英文自在。
  • 7月 辞职，不得已的选择，辛苦宜生同学替我办了几乎所有的手续。去新加坡的飞机上，脑海里一直浮现着八个字：广阔天地，大有作为。💪
  • 8月 全身心地写论文，很顺利，很充实，很开心。这阵子爱上一首翻唱歌曲—《野孩子》，30号那天循环播放了95次。
  • 9月 参加了NUS GivingRUN 🏃，新加坡天气热，动不动就一身汗。5公里感觉完全不虚，明年可以报10公里，毕竟是半马过的人。
  • 10月 很开心地把这里和美国的导师撮合在了一起，合作完成了GAN-Airfoil论文。这项工作虽然耗时不长，但却最令我骄傲。和办公室的同学一起去马六甲玩了两天。和NTU的一个朋友交流了一下，NTU校园比NUS漂亮。
  • 11月 GAN-Airfoil论文投了。学会了周日躺尸一天。
  • 12月 这个月新加坡阴雨连绵，气温宜人。周末和两个朋友去乌敏岛骑行，晚上制作了vlog，欣赏一下，觉得很好玩。

取得的成绩

1. 基本满足了科研任务
投了三篇论文，两篇已发表，另一篇在修改; 写了四个申请，中了两个。

2. 找到了海外博后
我的研究课题只在中国和美国比较受关注，排除这两个地方，不容易找。
多谢自己没放弃，多谢大云提供信息，多谢老师信任。

3. 快速适应了新的生活
很佩服自己的选题

不足的地方

1.工作状态的稳定性不够
认真工作时候，不想任何人打扰，这是好习惯，保持。但对待家人朋友要多点耐心。

2.缺乏规划和总结
总结让人清楚从哪里来，规划告诉你要到哪里去。
不要吝惜这点时间。

3.缺乏和朋友的深度交流
交流很少，没有榜样，进步很小。

4.没有早点换住处
和房东一起住杂事比较多，既然不喜欢，为什么不早点走呢。

学到了什么？

1.不要逃避应有的孤寂
找工作期间，对未来没希望，对自己不信任，内心很煎熬。
可谁又不是这么过来的呢？身边有两个朋友无业半年才有了选择，谁让我们有点追求呢 🏄
没有免费的午餐，面对看似很美的选项，要考虑自己能为对方做什么。

2.要有稳定的奋斗目标
年底的一个月丢失了目标感。
脑海里挥不去的一个想法是到这个年龄了还在孤身奋战，不是浪费生命吗?
是不是不知道，但这样的自怜的确浪费掉了一个多月的时间。
不管是怎么样的一生，肯定都有春夏秋冬四季变化，还是要做好当下的事情。
枯燥的时候，不要急，细细品，慢慢会有味道的。

3.融入当下的环境
到过哪些地方、看过哪些风景、遇到过哪些人、经历过哪些事情，这本质上在描述着你的一生。
不管以什么目的来到了这里，不要过客一样置身事外，爱上这片土地，爱上这些人，因为这里已无可选择地成了你的一部分。
大学四年没什收获吗，别怪母校，怨你没有爱上这片属于你的地方。

留美幼童

发表于 2019-03-17 | 分类于 感想

  • 容闳：1854年毕业于耶鲁大学，”大人者不失其赤子之心”。伟大之人必有伟大之志，历时18年策划并实施中国派遣幼童赴美留学计划。
  • 1872年-1875年派出4批共120名，到1881年原定十五年的幼童留美计划中途夭折，尚存的94名学生全被召回国。
  • 1872年8月11日：第一批30名赴美幼童派出，平均年龄12岁，无一贵胄子弟。所签15年留学协议：“业成后回华差遣，不得私在华洋各处私谋生理。其在洋在途，如有天灾疾病不测等事，各安天命。”
  • “洋人之所长在机器，中国之所长在人心。照洋务派的做法，就会抛弃礼义廉耻，把洋人的学问当成宝贝，人人争相崇洋，洋枪洋炮有了，可谁愿为国献身呢？” 变革的难处在于：反对者的担忧在局限的条件下是成立的。每一种坚持的背后，其实都有站得住脚的理由。
  • 人数不多、中途夭折，却依旧成就了近代铁路、电报、外交等领域的脊梁，功不唐捐。
