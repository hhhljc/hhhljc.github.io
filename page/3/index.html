How to make things work?

发表于 2018-08-23

俞敏洪爬悉尼大桥过程中对创业成功要素的总结：

  1. 一个直击痛点的好点子
  2. 一个对这个点子极其热爱的人
  3. 着力解决这个点子可能带来的负面效应（安全问题，环保问题）
  4. 利用名人带来爆款效应
  5. 赢得政府支持
  6. 创业者的坚持
  7. 围绕该项目的持续优化

摘自 在人生的更高处相见

这与学生时期写论文的感觉非常相似。除了第6点是所有事情都应具备的条件，第4，5点对于学生搞学术并不重要之外，其余4点的确是PhD做学术成功的关键（目前仍这么认为 2018-08-23）：

  1. 研究对象应为行业痛点 (什么阻碍了数值优化在工程中的应用？)
  2. 对提出的解决方案痴迷 (痴迷之前不要动手)
  3. 在实现该方案的过程中发现并解决问题 (Mode, ME, Smoothing…)，这个过程中应坚持尊重事实的原则。很多问题都不是想当然的那样，尊重事实才能彻底解决问题。
  4. 持续测试、改进 (more and more papers)

一周奥兰多

发表于 2018-01-17

AIAA会议

我不是外向的人，但我需要适应做报告，所以要努力。
事实上，在去之前我花了大量的时间练习，临行前的组会，老师照例让每个人都做了演练，相应提出改进意见，后来还专门发邮件对slides做了具体修订。
当时主要的问题有三点：

  1. 英语表达仍然欠缺
  2. 不够自信，需要更频繁地与观众进行眼神交流
  3. 肢体语言不够。

总结
并没有拿到最佳论文，不重要；最欠缺的是socialization，本应做得更好。
期间有各地的牛人，还有一些非常有意思的报告。
听了很多场，却没有一次提问，几乎也没有跟作报告的人进行交流，只是去找了一个国内来的学弟瞎聊了几句没营养的。
事实上，其中的一个报告者就是我后来一篇文章的审稿人，他给了一些很有建设性的意见。
虽然文章后来被顺利接受，但当时能去和他聊一下自己的想法，这个过程必将会让自己对social的认识更为积极。
而这种认识是必需的。
