《文城》

发表于 2021-07-04

从许三观开始偏爱余华。

林祥福是本分善良的年轻未婚少东家，自称来自文城的女子出现又离开，带走了世代积攒下的一半黄货，勾走了林祥福的魂。林变卖家产，背上襁褓中的女儿，抱着赴死的决心开启寻妻之路。文城是个假地名，林将自己的后半生停在了一个他觉得最像文城的地方，与一群本不会有交集的人深刻地发生起了关系，直至付出生命。林是个过于勇敢的人，不甚幽默变通，所以走出的路显得异常坚毅。作为读者，不禁会问值得吗？

“生活没有眼睛，欺骗是它的导盲犬。”不知道值不值得，但生活骗你往往没商量，我们一时很难有别的选择。身处文城，他肯定也知晓这些，但也没有表现出过多痛苦，难道是他不懂变通吗。事实上，即便人生轨迹发生了这么大变化，熬过最难的那些日子，他即刻又在千里之外开启了祖辈的生活模式，而且快速壮大，恢复了元气。所以被骗并没有那么可怕，即便是发生在重大人生道路的选择上，不过是让我们换个领域换个地方发光发热而已。环境很重要，更重要的还是自己要做个什么样的人。

晚上看完了文章补集，俨然另一个视角的故事，故事性比正文要强点。小美是好人，阿强也是好人，这个社会好像没有坏人，可是大家仍旧过得好生凄惨。好人也会欠债，好人也得还债。

“顾家四兄弟撑杆跳翻墙去嫖那段把我笑死了哈哈哈哈”，的确是，哈哈哈哈哈。

《秋园》

发表于 2021-06-24

太心疼作者了，想说三句

  • 去他妈的贫困
  • 去他妈的愚昧
  • 去他妈的窝里斗

这样的命运，该怎么生活下去？假如生活欺骗了你的最后一句是“而那过去了的，就会成为亲切的怀恋”。故事的悲凉在于好多人都死了，这也是许三观比活着温暖的原因。不要怕不要急不要死。

作者：李青菜（来自豆瓣）
来源：https://book.douban.com/review/12602283/

吾土吾民，我们怎么这么倒霉，几十年如一日，几百年如一天，有皇帝苦没皇帝也苦，孩子读不上书，大人不如牲口，四老倌的盐、兵桃的蚂蟥、小泉的布料、满娭毑和二菊、张跛子的黄草纸、人王的杂耍，王成恩的农药……贫穷惨烈、荒诞愚昧、自相残杀的故事还在我们身边，从不落幕。

《装台》

发表于 2021-01-19

上周看了电视剧，周末又花了一天多的时间读了小说。

先说说电视剧，我觉得闫妮演得不好，也不是，可能是这个角色不光鲜，所以不喜欢。张嘉译演得顺子不太像个小人物，其实也不是小人物，包工头嘛。小说里形象更清晰一些，顺子更“猥琐”怯懦，蔡素芬更鲜活一些，敢爱敢恨。

整体上小说传达的意思更清晰，电视剧改动虽不太大，但结尾一变整个意思也就变了，小说里顺子最后坚定的点头太伟大了。人生负重前行，虽战战兢兢而意外遭遇不断，哪有什么大团圆，还能做到虽千万人吾往矣太牛逼了。

印象较深的是对蚂蚁的描写，秩序井然、分工明确、不想那么多。顺子做梦变身蚂蚁，问浩浩荡荡的蚁群要去哪里，一只忙碌的蚂蚁惊讶又气愤得说“这是你该考虑的事情吗？”真是好笑。蚂蚁的社会当然可以这样，整个蚁群才像一个整体，人也不是每个细胞都要思考人生。

《我是我的神》

发表于 2020-11-24

应该是今年读过的最厚的一本书，有80万字，花了好几天才看完。最初是被标题和一个豆瓣书评吸引，以为一定是个主宰人生的故事，没想到里面的人过得非常拧巴。我并不太理解乌力二代们的纠结，尤其是天赫，怎样的仇恨才能离家20年。很多人说这是个好故事，读后醍醐灌顶，我却并没有太多感受。读之前，人生意义的问题已经在脑海中缠绕了两个月，读完依旧不清楚。

乌力二代们的童年应该算很幸福了，父亲是军中的司令员，在那个吃糠咽菜的年代绝对是吃穿不愁。可以说，现代很多年轻人努力奋斗的动力就是能让自己下一代过上乌力二代们的生活。然而，在这样的优越环境中长大的乌力们却以极其叛逆的色彩度过了半生。乌力一代想不明白，都革命了，自己的孩子怎么还过得不好；如果我们下一代还如此纠结，那么努力奋斗还有意义吗。

回味一下，故事确实挺精彩的，这几天都不怎么看视频了。仔细想想书中各人的生活，又觉得书名取得恰如其分，保持勇敢、保持善良，我是我的神。

无问东西

发表于 2020-10-04 | 分类于 感想

下午看了《无问西东》，比想象中的要好。该如何度过一生，这种话题不好讨论，通过传承不断强化论点，煽动性好强。电影将追求真善美的阻力定义为世俗，而世俗后面是家人朋友甚至目之所及的整个世界，很难克服。人生就是不断权衡的过程吧，所以才要不断思考生而为人的所谓虚头巴脑的意义，构建起自洽的人生观，才能对抗或者顺从“世俗”。

三十岁之前的日子过完了，什么感觉呢？想起一句相声台词，“我还是那个说相声的小学生”。不是开玩笑，当下的状态是几周就能自我怀疑一下子，三观大概有个样子但也讲不清楚，对事物判定不清、容易摇摆。这个样子下去也不知道到40岁能不能小学毕业。

如果给二十岁一个建议的话，我觉得应该是要“不怕”。有想法就勇敢追求，不给自己退路。否则会有太多时间浪费在担忧上或花在为失败做准备上。二十几岁的年纪没有负担，只要不自暴自弃，怕什么，条条大路通罗马。

今后怎么过呢？也说不清楚。不过还好能够勉强做做短期的规划了，不偷懒的话，应该会不断成长。也渐渐觉得自己对家人应该承担更多，否则我想趁年轻多流浪几年，欧洲我还没去的嘛。

《许三观卖血记》

发表于 2020-09-07

好希望每一个生命都如此充满韧性。

许三观的性情决定了肯定会让一乐去喊魂，余华写到这里就任由角色自由发挥了吧。能用嘴给全家人炒菜，得是个多么暖心的人。看这段不由得想起了小时候一家人躺在一张大床上听我爸讲故事，嘿嘿嘿。三观夜宿客栈，大爷分给他一直猪娃儿暖脚，这一段能让人笑出眼泪，一定是余华或者他身边人的经历吧。我爸讲他小时候每次玩到后半夜回家都会钻到我二爷的被窝里，听着就暖暖活活的。真是一部好看的作品。

《活着》

发表于 2020-09-05

电影本科的时候看过了，架不住推荐评价很诱人，“看完通透的大哭了一场”。今天雨淅淅沥沥下了一整天，心情也比较失落，觉得能一口气看完个故事也很好。的确挺好的，大概4个小时读完了，中间小规模落过几次泪，不甚汹涌，不过心情总归通透了些。

先大致说一下几个印象：

  1. 富贵和苦根的对话像是谈恋爱，苦中作乐。
  2. 因为看过电影，人物很熟悉，看到家珍总是想到巩俐，演得真好。
  3. 村长哄走众人讨要粮食的话语好诚恳，村长后来对富贵家高低不错，多少是念了这一口粮的恩情吧。
  4. 富贵在和牛谈恋爱。我们生来是要和自己的世界恋爱的，浪漫、风趣、苦中作乐、相看不厌。

总在思考怎么活才不荒废，痛苦焦虑以至无法入睡，而有些人却面对的是怎么才能活下去。一直以来，我觉得自己的思想都不够自洽，满意的生活工作状态不出数月总要宕机。生活工作谁能保证一帆风顺呢，世间事大多是计划赶不上变化。

我的世界是家人，朋友还有工作，我要善待你们。

《天生有罪》

发表于 2020-08-19

这个母亲好伟大。独立自主、有追求、自信坚定、乐观幽默，作者很少直接夸她，但字里行间都是对母亲的感激。也有缺点，比如痴信上帝，他们的生死经历大多和这有关；对爱情或者婚姻家庭关系的处理失当，为什么选择那厮、为什么不离开他、为什么还要生三胎。。。这些多多少少也和时代有关吧。人无完人，如此伟大的母亲也走了这么多弯路。不过也不妨碍啊，“上帝”还是派来了付账单的使者不是吗。

最重要的是要有一个自洽的处世原则，或者说三观吧。他妈妈把这些都交给了自己理解的教义。文青们都觉得把信仰交给宗教有点太草率了，但建起完善自洽的三观并坚定准守之有多么不易，所以他们要折腾不休。我是愣头青，比文青还差点，还有很长的路走。

对子女教育的启示

  1. 惩戒不建立在发泄情绪之上。惩戒是为了让他知道做出错事情就会得到惩罚，这是这个世界的规则，世界不爱你，但我还是爱你的，所以惩戒过后我们还可以一起看球。
  2. 面对无奈的困境，还可以选择幽默。事情过了之后，你会发现这种选择有多美好。巴菲特说过，但凡没有死人的挫折都会被证明是好事情。是的，我们束手无策，但还可以一笑而过。
  3. 勇敢勇敢勇敢

《活过，爱过，写过》

发表于 2020-08-04

李银河的自传，前半部分读起来很像另一个人，不过李的文笔要差一些，那个人心思更细腻一些。

之前大概知道李银河这么个人，谈性谈爱谈王小波，尺度挺大。一直以为李王之间是某种为世俗不容的关系，比如年龄差距很大比如第三者插足，不然何须那么激进，没想到竟几乎算是从校园爱情走过来的明媒正娶。李年轻的时候挺好看的，王长得不行，刚开始李嫌弃王，两人因此闹过分手，哈哈哈，哎世人口中的完美爱情哈。

李很爱王，出于对王才华的欣赏，李甚至觉得可以独自工作挣钱让王坚持写作。很羡慕这样的眼光和勇气，我向来不自信，不觉得自己能做成什么事情，因此对另一半有诸多苛求，比如你要好好学习、好好工作、去替下一代看看世界更好。当然也一直缺少安全感，像用一份奖学金让对方陪读的事情我是断然不敢做的，疯了吧，你以后怎么找工作。王在美国陪读的几年应该是他们最开心的日子吧，毕竟他们相识相伴也只有二十年。看着那些去过和没去过的但都熟悉的地名，好生羡慕。

他们回国后九年，王突然去世。李的文笔缺少细节描述，不知道她如何过渡过了那段时间，每天梦醒的时候会不会绝望到窒息？然而，三个月后李就在某种派对上认识了余生的伴侣，面对这家伙的猛烈攻势，李从了。我很难理解这样的故事，她和小波如此真挚的爱情不需要时间整理吗？

后一半讲了她的学术研究，不好看。她很少提那个伴侣，只说爱上了写小说，除去吃饭睡觉每天有14个小时可以写小说，好幸福。我觉得她一定不如爱小波一样爱这个家伙吧。她提冯唐很多次，我没看过冯唐的东西，隐约感觉有个词叫油腻跟他有关。李最后写了人生观，蜜蜂采蜜理论，人生的意义在于那些好吃的好玩的，大概就是及时行乐吧，这也解释了那个选择。

西海岸公园

发表于 2020-07-29

自从伤了膝盖后，上厕所蹲下站起都费劲，将近一年没有跑步了。最近在看一本关于跑步的书，渐渐又勾起了跑步的欲望。傍晚下起了小雨，终于没忍住下楼拐弯去了西海岸公园。

其实从来也没有坚持过跑步，不然也不会跑伤膝盖，但我有一个屡试不爽的体验：不管多么无聊颓废，拉出去跑个五公里，即刻热能爱起好吃好喝安静读书的生活。果然饱暖思淫欲，小资才爱伤春悲秋。

西海岸公园是块狭长的绿地，去年十月第一次去散步，惊讶这么拥挤喧嚣的地方附近还有如此静谧的去处。年初搬来这边居住，时常到这个公园散步+思考人生🤔，收获颇丰。但今天跑起来却没多少想法，半程过后只想尽快跑完回家洗澡，配速超了6分半，大概属于老奶奶级别。果然我们不一样，人家跑步能出本关于人生的书，我只能跑湿T恤。
