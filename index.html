《我不是潘金莲》

发表于 2022-01-06

主线事先都知道了，但故事依旧非常好看。好看的小说一定写的不是生活化的故事，但人物心理一定贴近生活。

刘震云小说好看的一个重要原因，是可以用反反复复的啰嗦复现出人物的当时心理活动。比如李雪莲被大头骗后没有第一时间分析这件事情的窝囊，而是直到几天后情绪崩溃的时候才一并写出，显得非常自然。

罗永浩调侃库克

发表于 2022-01-03 | 分类于 老罗

今天看到老罗在新年第一天把苹果CEO骂了个狗血淋头，觉得哈哈哈，老罗应该没事儿（不会像薇娅那样），还可以看到“真还传”，期待。

喜欢老罗，能够戏谑又诚恳地把矛盾讲清楚。他讲自己打磨宣传海报流露出地细节控，听起来让人会心一笑。但《我的奋斗》写得太敷衍，是演讲稿改过来的？即便如此，还是非常喜欢里面的一句话：

“如果你一生耿直，没有做任何恶心的的事情，没做对别人有害的事情，一辈子拼了老命顽强把自己身边的几个人照顾好了，老婆孩子老娘照顾好了，没有成名没有发财，没有成就伟大事业，然后一生正直，到了七八十岁梗着脖子去世，你这一生是不是没有改变世界？你还是改变世界了，你把这个社会变的美好了一点点，因为世界又多了一个好人。”

《一句顶一万句》

发表于 2021-12-18

小说本身比较奇特，心里活动很多所以很有沉浸感，有序的展开混沌的生活。世间的事情你连我，我连着你，不是一个理能说清楚。就本书来看，我感觉它和其他小说都不一样，但也不乏贬低的读者，各有各的理吧。人物众多，大多以“老+姓”相称，很是亲切。能说得上话的人，不一定非多能说，就图个你情我愿，对脾气。找对象也得是这么个人，愿意一起“喷空”。

“不要说精神的痛苦只有知识分子才有”，可不是嘛。

《我的奋斗》

发表于 2021-10-18

只能说写得非常简单，不过从老罗看似非常确定性的表述当中，可以看出“读到”、“想到”与“做到”的区别。对于影响生活的重大问题，比如迷信、贞操、歧视等，突然觉得好像可以想清楚并坚定执行之。罗在30岁或者更小的年纪就搞得很明白，并且是那种不丧失敏感的粗厚，仰慕。

《故乡天下黄花》

发表于 2021-10-16

今天周六，早上看到了这本小说的简介。之前读细雨和人生海海的时候，意识到自己对乡村生活由衷亲切，刘是河南人，写故乡，我循着这个心思选了这本书。

整本书写了辛亥、抗日、土改、文革四个时期，看完前三个实在提不起心气再读下去，这不是我心中的故乡，我不愿意承认。与改朝换代同时发生的是礼崩乐坏，大乱后的大治需要很久的时间，久到需要慢慢等一代代人老去。悲观地想，要等每个人都有了科学的价值观、对事物有客观的认识、可以尊重不同意见的时候，人才会是人。

看到一些更为精确的表达：

  • “成长环境告诉我，有些生活工具和方式，从农耕跨到信息社会过了一百年他都没变过丝毫。”
  • “同时作品冷漠与阴暗的风格特点更来自于作者对人性的绝望，作品虽然只讲述马村半个多世纪的人事变迁，但重复的故事情节使读者感到千百年如一日，人性如此，凝滞不变。”

《人生海海》

发表于 2021-10-04

本来觉得今年看过的最佳小说应该是余华的细雨了。

坚守、犯错、逃避、认罚、漂泊、放浪，人生海海，有多少苦难是因为旁人的看法而引发。这个世界上坏人挺多，因为一不小心我们自己也会做出坏事。不要让自己沉溺于坏人坏事，大鹏一日同风起，乘风破浪会有时。

我发现自己对乡村的那种八卦生活很亲切。

  • 麦家对人生海海这个词的理解：既然每个人都跑不掉逃不开，那不如去爱上生活。

  • 莫言的评价：小说的迷人之处就在于它能把不存在的人物写得仿佛是我们的朋友，麦家的《人生海海》就是这么迷人。的确如此，书中的主要人物形象那么近，又有些许神秘。

  • “人生海海，错了可以重来。一辈子那么长，总会做几件错事，做一些让自己后悔一生的事情，也许是高考那年填错了志愿。也许是在女朋友需要你时，没有挺身而出。无论在什么样的年龄，犯了什么样的错，但只要回头，就一定有重来的机会。尽管有许多事情错过了，就没办法再来一次，就像我后悔高考没有填报文学专业一样。但无论是犯错、还是错过，并不代表彻底失去了希望。当一个小说家最好的时间是十年前，其次是现在。”

  • 既然逃不掉，就好好地感受它吧！正如《人生海海》中的上校一样，经历了一生，仍然了“疯”的方式活成了自己。“小瞎子”经历了人生后，依旧以“瞎话”继续苟活。

《一个陌生女人的来信》

发表于 2021-08-21

看了两个故事后，才发现这是一本短篇小说集。
还有印象的几个故事：

  1. 小男孩与妈妈追求者的战斗 （灼人的秘密）
  2. 失明的画册搜藏家老者 （看不见的珍藏）
  3. 旧书贩门德尔 （旧书贩门德尔）
  4. 暗恋毕生的女人 （一个陌生女人的来信）
  5. 和自己下棋的人 （象棋的故事）

故事都不大，精彩性来源于详细的心理和动作描写。失明老收藏家的生活令人可悲可喜；陌生女生的来信，是多么“病态”的爱情才能耗掉整个生命？

读这本书有一点点思想转变，无须也不可能从每本书中总结提取人生法则，阅读具有的镇静功能已经很受用了，给彷徨以希望，给失败以慰藉，给孤单以陪伴。

《看见》

发表于 2021-07-25

最近一周看了这篇柴静关于自己在央视十年的新闻回忆录。虽然花了一周才读完，过程挺匆忙的，因为心里有事儿。读完还是有哪种依依不舍的失落感，像小时候看完电视剧一样，难过“他们都离我而去”。

柴静前些年被黑的很惨，主要有两点：

  1. 13年赴美生女
  2. 关于碳排放权采访丁仲礼的那期节目，做了苍穹之下竟然还能问出这么无知的问题。

我的看法是

  1. 公众人物也有的个人权利。
  2. 采访不是辩论，作为记者问出的问题不代表个人观点，不然央视怎么可能播出呢？这期节目最重要的一点是让普通百姓深刻地认识到西方的伪善和大国竞争的残酷程度。因此要警惕民粹。

柴是一个认真的工作狂，或者说是很能享受工作的人，不然不会从专科一路上来最终到央视。《看见》一本书中柴记者以诚恳地态度讲述了自己十年新闻记者的心路历程，讲述了从感性地抒发情怀到严谨地格物求真的转变，无奈大众情绪看不得真正的人。

这本书读的时候我标注了很多

  • 情感不是爱。哭和笑只是情感上的反应，爱是要付出行动的。情感是爱的前提，但也没什么值得得意的，离爱人还很远，想到和做到的距离。所以从INTJ到INFJ又能怎么样呢，重要的不是想到，是做到。
  • 事情只有认真对待才有收获。一次演讲比赛。
  • 我们终将浑然难分，像水溶于水中
  • 宽容的基础是理解，你理解吗？宽容不是道德，而是认识。唯有深刻地认识事物，才能对人和世界的复杂性了解和体谅，才有不轻易责难和赞美的思维习惯。
  • 你想采访弱者，就要让弱者同情你。
  • 痛苦是财富，这话是扯淡。姑娘，痛苦就是痛苦，对痛苦的思考才是财富。
  • 我不是要告诉你怎么改，我是要激发你自己改的欲望。这话可以转给老板。
  • 海子有句诗，深得我心：”天空一无所有，为何给我安慰。”
  • 张北川：”我们的性文化里，把生育当作性的目的，把无知当纯洁，把愚昧当德行，把偏见当原则。” 又岂止是性文化。
  • 厄运中的人多有一种对自己的怨憎，认为是自我的某种残破招致了某种命运。
  • 斯宾诺莎：”嘲笑、轻蔑、愤怒、报复……这些情绪，都与恨有关或者含有因恨而起的成分，不能成为善。”
  • 原来大家童年都寂寞。

豆瓣热评：

  • 字里行间，满满的自恋。自恋有什么不好？
  • 柴静的优点就是她不断在反省。格物致知不是本书的核心吗
  • 急急地读完了，想法不少，明天再细说吧。只觉得人活一世真是宛如盲人摸一头奇大无比的象，在最初面对奇观的惊讶之情褪去后，只会发现自己既不能解释，也无法否认。只有终其一生触摸下去。书名也就是此意吧。的确如此

《在细雨中呼喊》

发表于 2021-07-09

打完第二针疫苗，收拾利索坐等发烧，两天看完了这部散文体长篇小说。

故事夹叙夹议讲述了一个农村贫困家庭孩子孙光林的青少年时期。孙光林是哥仨中的老二，由于被送养五年又返回的经历，孙光林在贫困的家里极不受待见，使其性格极其敏感。借助这双敏锐的眼睛铺开了少年经历的一系列故事，作者似乎在传递一种观念，我们本质都是朴实善良的，所有的选择背后有着穷困的因素。整个故事的背景虽是60-70年代，却故意淡化了时代背景，读起来非常亲切。看到一条短评，我深表同感：这本书试图表达人们在面对过去时，比面对未来更有信心。因为未来充满了冒险，充满了不可战胜的神秘，只有当这些结束以后，惊奇和恐惧也就转化成了幽默和甜蜜。这就是人们为什么如此热爱回忆的理由。
